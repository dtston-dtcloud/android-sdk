# Android SDK
## 简介
深智云的SDK封装了手机与深智云硬件，以及手机与服务器通讯的过程，这使得开发者可以更快速的完成APP开发，避免了面对复杂的协议和错误处理，只需要专注于APP用户体验的优化和设计。
